# Buscador de Películas 🎬

Aplicación web desarrollada con Flask siguiendo arquitectura MVC.

## Funcionalidades

- Buscar películas por título y año (OMDb API)
- Ver detalle de la película
- Agregar comentarios por película (SQLite)
- Visualizar comentarios guardados

## Tecnologías usadas

- Python
- Flask
- SQLite
- OMDb API

## Instalación

1. Clonar repositorio
2. Instalar dependencias:

pip install -r requirements.txt

3. Ejecutar la aplicación:

python app.py
